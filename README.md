# Generated Gitlab graphQL api TypeScript types

Gitlab type definitions generated from Gitlab graphQL api schema

## Installation
```
npm install -D gitlab-graphql-types
```

## Conusming
```
import {Project} from "gitlab-graphql-types";
```
